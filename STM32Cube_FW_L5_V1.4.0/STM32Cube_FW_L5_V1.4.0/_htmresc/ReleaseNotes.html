STM32CubeL5 Firmware Package V1.4.0 / 10-February-2021


Maintenance release of STM32CubeL5 Firmware
  • Corrections and enhancements of CMSIS Device, HAL and Low Layer drivers
  • Middleware updates: FreeRTOS, USB Device, USBPD Core and Device
  • Renamed Middlewares/mbedTLS path to fix Linux build
  • ICACHE peripheral enabled in projects to get a 0 wait-state execution from Flash memory and external memories
  • Project examples updated to STM32CubeMX 6.2.0
